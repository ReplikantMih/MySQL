Что за база данных:

База данных `upwork` реализаует хранение и работу с данными для реализации
основных механик для сервиса аналогичного сервису UpWork.com.



Про что эта база данных и сервис:

UpWork (и, соответственно, база данных) - сервис, предоставляющий возможность
заказчикам реализовывать свои проекты, а фрилансерам искать работу и получать
вознаграждение за выполненную работу. 

Представляемая база даных описывает следующие основные сущности (подробней
 в описании при создании таблиц):

- freelancer - фрилансер, лицо, выполняющее работу по проекту.
- client - заказчик - лицо, описывающее требования к проекту, курирующее проект,
оплачивающее выполненную работу, выбирающее исполнителя (фрилансера).
- project - проект - объект, инкапсулирующий проект, конкретного заказчика,
исполнителя, их переписку, описание требуемых работ и результата и т.д.
- milestone - промежуточные отсечки по проекту.
- proposal - отклик фрилансера на проект (предложение себя в качестве возможного
исполнителя по проекту).
- invite - предложение от заказчика к конкретному фрилансеру стать исполнителем
по конкретному проекту данного заказчика.
- category - предметная область, к которой относится проект (аналитика, разработка,
дизайн, маркетинг и т.д.).
- account - внутренний денежный счет для ведения рассчетов между всеми участниками -
фрилансерами и заказчиками.
- payment - операция по счету (account), при которой меняется значение balance
(баланс) счета.


Представленная база данных призвана обеспечить выполнение таких действий и
операций как:

- создание проекта заказчиком;
- поиск проектов фрилансером по описанию, по категории, по цене, дате создания;
- предложение фрилансерами своих услуг в качестве исполнителя по конкретным
проектам;
- предложение заказчиком конкретному фрилансеру стать исполнителем по проекту;
- пополнение внутреннего счета фрилансерами и заказчиками;
- обмен внутренними сообщениями в т.ч. с прикреплением media документов;
- и т.д.
